# TripLog

[![CI Status](https://github.com/AY2526S2-CS2103-F13-2/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103-F13-2/tp/actions)
[![codecov](https://codecov.io/gh/AY2526S2-CS2103-F13-2/tp/branch/master/graph/badge.svg)](https://codecov.io/gh/AY2526S2-CS2103-F13-2/tp)

![Ui](images/Ui.png)

**TripLog is a desktop application for managing your travel experiences and itineraries.** Optimized for fast typists, it combines a powerful Command Line Interface (CLI) with a clean Graphical User Interface (GUI) to help you log destinations, track trip statuses, and organize memories with zero friction.

* If you are interested in using TripLog, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested in contributing to TripLog, the [**Developer Guide**](DeveloperGuide.html) provides a comprehensive architectural overview.
